━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[ 핵심 ]  아래 "정해진 파일 이름"으로 이미지를 이 img 폴더에 저장만 하면,
          해당 등급 운세가 나올 때 그 짤이 자동으로 카드 위에 뜹니다.
          (안 넣은 등급은 기본 짤이 그대로 유지됩니다.)

[ 등급별 파일 이름 ]  확장자는 .png / .jpg / .gif 아무거나 OK

   초대박(완전 럭키비키)  →  my-jackpot.png   (또는 .jpg / .gif)
   대길(가보자고)         →  my-daebak.png
   길(개이득)             →  my-good.png
   평(그럴 수 있어)       →  my-soso.png
   소흉(어우야)           →  my-oops.png
   흉(오늘만 버티자)      →  my-ugh.png
   대흉(오늘은 글렀다)    →  my-doom.png

[ 인터넷 짤 저장하는 법 ]
   1) 브라우저에서 원하는 짤 이미지 위에 마우스 오른쪽 클릭
   2) "이미지를 다른 이름으로 저장"
   3) 저장 위치 = 이 img 폴더,  파일 이름 = 위 표의 이름 (예: my-doom.gif)
   4) 사이트(localhost) 새로고침 → 끝!

[ 주의 ]
   * 파일 이름을 정확히 위와 똑같이 (영문 소문자 + 하이픈). 한글 이름은 피하세요.
   * 배포(Netlify Drop)할 때는 'deploy 폴더 전체'를 올려야 짤도 함께 올라갑니다.
   * 한 등급에 짤을 '여러 장 랜덤'으로 넣고 싶거나, 위 방식이 어려우면
     파일만 폴더에 넣고 저(Claude)에게 말해주세요. 코드로 직접 연결해 드립니다.
